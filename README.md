# Buffett Investment Analyzer

## 概要

ウォーレン・バフェットの投資哲学をAIで再現し、
企業情報を分析して投資判断をサポートするWebアプリです。

---

## Version

現在開発中

Version 0.3.0

---

## Version 1.0で実装予定

- Buffett Score
- 日本株分析
- 米国株分析
- AI定性分析
- PDFレポート

---

## 使用技術

- Python
- Streamlit
- yfinance
- OpenAI API
- pandas

---

## 起動方法

```bash
pip install -r requirements.txt

streamlit run app.py
```
